## Key Features

### Supported Devices

* RZ/G2L  Evaluation Board Kit (RTK9744L23S01000BE)
* RZ/G2LC Evaluation Board Kit (RTK9744C22S01000BE)
* RZ/G2UL Evaluation Board Kit (RTK9743U11S01000BE)
