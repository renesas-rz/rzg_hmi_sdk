## *RZ/G HMI SDK v3.4.0.0*

{% endif %}

2026/01/30
{: .right .small }

RZ/G HMI SDK version 3.4.0.0 is available.

- Added support for the **Flutter** UI framework (v3.27.1).
- LVGL has been Updated LVGL from v8.3 to v9.2.2.
- Updated Chromium from v109 to v132.
- Added support for the **RZ/G3E EVK**.
- Updated Yocto from Yocto 3.1 (dunfell) to **Yocto 5.0 (scarthgap)**.
- Updated Linux kernel from 5.10 to 6.1.

!!! warning "Notice"
	Please note that HMI SDK v3.4.0.0 supports only the RZ/G3E EVK.
	Support for the RZ/G2L, RZ/G2LC, and RZ/G2UL EVKs, skipped in this release, will be added in v3.4.1.0 (<span style="color: var(--renesas-secondary-color-fg-red">coming soon</span>).
	Until then, use the [HMI SDK v2.3.1.0](https://renesas-rz.github.io/rzg_hmi_sdk/v2.3.1.0/) for the RZ/G2L, RZ/G2LC, and RZ/G2UL EVKs.
