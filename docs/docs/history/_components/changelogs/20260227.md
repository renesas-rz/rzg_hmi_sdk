## *RZ/G HMI SDK v3.4.1.0*

{% endif %}

2026/02/27
{: .right .small }

RZ/G HMI SDK version 3.4.1.0 is available.

- Added support for the **RZ/G2L EVK, RZ/G2LC EVK, and RZ/G2UL EVK**. <br>
  HMI SDK v3.4.0.0 introduced a new UI framework (Flutter) and updated the Linux kernel, Yocto, and existing frameworks; however, it supported only the RZ/G3E EVK.
  HMI SDK v3.4.1.0 adds support for the RZ/G2L, RZ/G2LC, and RZ/G2UL EVKs.
  As a result, the following updates are now also available on these EVKs:
	- Support for Flutter v3.27.1, LVGL v9.2.2, and Chromium v132  
	- Support for Yocto 5.0 (scarthgap)  
	- Support for Linux kernel v6.1

!!! warning "Notice"
	Please note that HMI SDK v3.4.1.0 does not include packages for the RZ/G3E EVK.
	If you would like to use the HMI SDK for the RZ/G3E EVK, please refer to [the webpage of HMI SDK v3.4.0.0](https://renesas-rz.github.io/rzg_hmi_sdk/v3.4.0.0/).
