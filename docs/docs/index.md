# HMI SDK for RZ/G Series

The RZ/G HMI Software Development Kit (HMI SDK) is an **all-in-one** Linux-based development environment for the RZ/G Evaluation Board Kit (EVK).<br>
{: .subtitle-normal .center }

It includes all the software required to build HMI applications quickly and efficiently.<br>
{: .sub-subtitle .center }

<br>

**Easy-to-Use**

<ul style="margin-left: 2em;">Provides a simple development environment, making it easier for Linux beginners to work with Linux MPUs.</ul>

**Free of Charge**

<ul style="margin-left: 2em;">All software included in the HMI SDK is available for free.</ul>

**Ready-to-Use**

<ul style="margin-left: 2em;">Includes the following commonly used GUI frameworks and libraries for HMI application development:</ul>

<ul style="margin-left: 3em;">
  <li>LVGL – A lightweight, open-source graphics library for embedded systems.</li>  
  <li>Chromium – A browser-based framework with GPU acceleration for application development.</li>
  <li>Flutter – Rich UI and UX like mobile apps and web apps.</li>
</ul>
